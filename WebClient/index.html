MailSender Backend Client Demo

Wyślij wiadomość przez MailSender API

Prosty klient demonstracyjny korzystający z endpointu /mail/send. Wklej token JWT wygenerowany przez /client-app/register, uzupełnij dane wiadomości i wyślij e-mail.

Formularz wysyłki

Dane zostaną wysłane do backendu jako JSON z nagłówkiem Authorization.

Jak testować?

Najpierw wygeneruj token w Swaggerze, a potem użyj go tutaj.

1. Rejestracja aplikacji Wywołaj endpoint POST /client-app/register i skopiuj wartość pola key.
2. Token JWT Wklej token do pola JWT Token. Skrypt doda nagłówek Authorization: Bearer.
3. Reguły biznesowe Temat zakończony znakiem ? dostanie prefiks [Q], a nazwisko w treści zostanie oznaczone markerem [student.surname].
4. Wysyłka Backend przekaże wiadomość do skonfigurowanego providera, np. Brevo.
